d:T2b80,<h2>What is HTML?</h2>
<p><strong>Did you know?</strong></p>
<p>HTML is the backbone of every website you visit? It's the magic code that structures web pages and makes all the content come to life!</p>
<p><strong>HTML stands for Hypertext Markup Language</strong></p>
<h3>Overview</h3>
<ul>
<li>
<p><strong>Markup Language</strong></p>
<p>HTML is a markup language that uses tags to define elements within a document. These tags, enclosed in angle brackets (&#x3C; >), tell the browser how to interpret and display the content.</p>
</li>
<li>
<p><strong>Document Structure</strong></p>
<p>An HTML document typically consists of two main parts: the head and the body.</p>
<ul>
<li>The <strong>head</strong> contains meta-information about the document, such as the title, character encoding, and linked stylesheets or scripts.</li>
<li>The <strong>body</strong> contains the actual content of the document, such as text, images, links, and other multimedia elements.</li>
</ul>
</li>
<li>
<p><strong>Tags and Elements</strong></p>
<p>HTML documents are constructed using a variety of tags. Tags come in pairs, with an opening tag and a closing tag. The content of the element is placed between these tags.</p>
<p>For example</p>
<pre><code class="hljs language-html"><span class="hljs-tag">&#x3C;<span class="hljs-name">p</span>></span>This is a paragraph<span class="hljs-tag">&#x3C;/<span class="hljs-name">p</span>></span>
</code></pre>
</li>
<li>
<p><strong>Attributes</strong></p>
<p>Tags can also include attributes, which provide additional information about an element. Attributes are placed within the opening tag.</p>
<p>For example</p>
<pre><code class="hljs language-html"><span class="hljs-tag">&#x3C;<span class="hljs-name">a</span> <span class="hljs-attr">href</span>=<span class="hljs-string">"https://www.example.com"</span>></span>Visit Example<span class="hljs-tag">&#x3C;/<span class="hljs-name">a</span>></span>
</code></pre>
</li>
<li>
<p><strong>Common HTML Elements</strong></p>
<p>HTML includes a variety of elements for structuring content, such as headings (<code>&#x3C;h1>, &#x3C;h2>, ...</code>), paragraphs (<code>&#x3C;p></code>), lists (<code>&#x3C;ul>, &#x3C;ol>, &#x3C;li></code>), links (<code>&#x3C;a></code>), images (<code>&#x3C;img></code>), and more.</p>
</li>
<li>
<p><strong>Version History</strong></p>
<p>HTML has evolved over the years, with various versions. HTML5 is the latest version and includes features for multimedia, form handling, and improved semantics.</p>
</li>
<li>
<p><strong>Web Browsers</strong></p>
<p>All major web browsers, including Chrome, Firefox, Safari, and Edge, support HTML. Browsers interpret HTML code and render it visually for users.</p>
</li>
<li>
<p><strong>Role in Web Development</strong></p>
<p>HTML is often used in conjunction with Cascading Style Sheets (CSS) and JavaScript to create dynamic, visually appealing, and interactive web pages.</p>
</li>
</ul>
