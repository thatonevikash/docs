d:Te17,<h2>Tags and Element</h2>
<p><em>In HTML every thing starts and ends with Tags</em> 😃</p>
<pre><code class="hljs language-html"><span class="hljs-tag">&#x3C;<span class="hljs-name">body</span>></span>
  <span class="hljs-tag">&#x3C;<span class="hljs-name">h1</span>></span>Hello World!<span class="hljs-tag">&#x3C;/<span class="hljs-name">h1</span>></span>
<span class="hljs-tag">&#x3C;/<span class="hljs-name">body</span>></span>
</code></pre>
<p>In the above block <code>&#x3C;h1></code> is a typography tag.</p>
<p>Which represents a heading text with most precedence.</p>
<p>In HTML, there are 6 typography headings with sequential precendence as
<code>&#x3C;h1></code> → <code>&#x3C;h2></code> → <code>&#x3C;h3></code> → <code>&#x3C;h4></code> → <code>&#x3C;h5></code> → <code>&#x3C;h6></code></p>
<p><code>&#x3C;h1></code> has the most precedence value and <code>&#x3C;h6></code> has the least precedence value.</p>
<h3>Tags</h3>
<p>As earlier I told that —
<em><code>"In HTML every thing starts and ends with Tags"</code></em></p>
<p>Alike <code>&#x3C;head></code> is a tag, <code>&#x3C;body></code> is a tag, <code>&#x3C;h1></code> is also a tag.</p>
<div class="alert alert-note"><div class="alert-header"><span class="alert-icon"><svg xmlns="http://www.w3.org/2000/svg" width="18" height="18" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" aria-hidden="true" class="alert-icon-svg"><circle cx="12" cy="12" r="10"></circle><path d="M12 16v-4"></path><path d="M12 8h.01"></path></svg></span><span class="alert-label">Note</span></div>
<p>HTML is all about dealing with various tags to build layout of any website.</p>
</div>
<p>A tag can only be written using <code>&#x3C;tag></code> angle brackets.</p>
<h3>Elements</h3>
<p>You may have noticed that there are pairs of tags as</p>
<pre><code class="hljs language-html"><span class="hljs-tag">&#x3C;<span class="hljs-name">body</span>></span>
  <span class="hljs-tag">&#x3C;<span class="hljs-name">h1</span>></span>Hello World!<span class="hljs-tag">&#x3C;/<span class="hljs-name">h1</span>></span>
<span class="hljs-tag">&#x3C;/<span class="hljs-name">body</span>></span>
</code></pre>
<p><code>&#x3C;h1></code> and <code>&#x3C;/h1></code> are the pair of opening and closing tags.
Which contains some text value.</p>
<p>Collectively, they are known as Element.
<code>One element = opening tag + content + closing tag</code></p>
<div class="alert alert-note"><div class="alert-header"><span class="alert-icon"><svg xmlns="http://www.w3.org/2000/svg" width="18" height="18" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" aria-hidden="true" class="alert-icon-svg"><circle cx="12" cy="12" r="10"></circle><path d="M12 16v-4"></path><path d="M12 8h.01"></path></svg></span><span class="alert-label">Note</span></div>
<p><br>
An element may or may not be pair of tags.</p>
<pre><code class="hljs language-html"><span class="hljs-tag">&#x3C;<span class="hljs-name">h1</span>></span>Hello World!<span class="hljs-tag">&#x3C;/<span class="hljs-name">h1</span>></span>
</code></pre>
<p>Is an element with pair of tags</p>
<pre><code class="hljs language-html"><span class="hljs-tag">&#x3C;<span class="hljs-name">img</span> <span class="hljs-attr">src</span>=<span class="hljs-string">"example.png"</span> /></span>
</code></pre>
<p>Is a self-closing tag.</p>
</div>
<p>Finally — <em>"An element is nothing just a meaningful tag."</em> 🤓</p>
