1b:Td95,<h2>Before Starting!</h2>
<h3>Where to start?</h3>
<p><strong>In the world of coding</strong>!
Where to start?, is the biggest question nowadays.</p>
<p>Many will suggest you, You can start coding in <code>notepad</code>.
Which may be a good idea but not the best.</p>
<p>Because, notepad is capable enough to make a source file.
Open notepad then create a file and save it with extension <code>.js</code></p>
<pre><code class="hljs language-txt">// index.js
console.log("Hello World!")
</code></pre>
<blockquote>
<p>[!important]<br>
<strong>source file</strong>, is the file which has an specific extension type.<br>
And when we exexute the file using a compiler, it produces some output.</p>
</blockquote>
<p><em>As I said</em>!
<code>notepad</code> is capable enough to create a source file, but it is not a best choice,
If you are targetting to code for longer.</p>
<p>You must go with an <strong>IDE</strong>! ⭐</p>
<h3>What is an <code>IDE</code>?</h3>
<p>IDE, stands for <strong>I</strong>ntegrated <strong>D</strong>evelopment <strong>E</strong>nvironment.
Which provides all <em>the facilities you will use to build any project</em>.</p>
