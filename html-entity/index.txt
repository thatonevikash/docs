1b:T6dd,<h2>HTML Entity</h2>
<p>Sometimes we use some special characters on a web page like angular brackets, copyright symbols, trademark symbols, etc.</p>
<p>But there is a problem with HTML we can't use angular brackets directly because HTML takes them as a tag, if we use opening and closing angular brackets together it will affect our code. To solve this bug we will use <strong>HTML Entity</strong></p>
<p>It starts with Ampersand (&#x26;) and should be closed with a semi-colon (;),</p>
<h3>List of HTML Entities</h3>
<pre><code>● &#x26;nbsp; - Non-breaking space
● &#x26;lt; - Less than angular bracket
● &#x26;gt; - Greater than angular bracket

● &#x26;amp; - Ampersand
● &#x26;semi; - SemiColon
● &#x26;reg; - Registered
● &#x26;trade; - Trademark

● &#x26;star; - Holo Star
● &#x26;starf; - Filled Star
● &#x26;phone; - Phone
● &#x26;male; - Male
● &#x26;female; - Female
● &#x26;hearts; - Heart
● &#x26;spades; - Spade
● &#x26;clubs; - Club
● &#x26;diams; - Diamond
● &#x26;sung; - Tune
● &#x26;check; - Check
</code></pre>
